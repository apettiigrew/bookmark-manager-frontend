# Bookmark Manager Frontend

Next.js 16 app with React 19 and Tailwind CSS v4, TypeScript.

## Commands

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run lint` — run ESLint

## Style Guide

All components must follow this design system exactly.
